About Dataset

MyDataset is a large-scale planning benchmark for intelligent agents. It features:

  • The world's first ML planning benchmark
  • 1200h of diverse sensor data across multiple cities
  • Multi-modal data (LIDAR, camera, GPS, IMU)
  • Detailed HD map and scene context
  • 3D bounding boxes labeled for multiple classes
  • Open/closed loop planning evaluation
  • 30+ realistic urban driving scenarios
  • Metrics for evaluating planners (safety, rule violation, dynamics)
  • Reactive + ML planner baselines
  • Free for research use
  • Contact us at mydataset@example.com for commercial use
